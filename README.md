# Raj Mahal Modern Hotel — Website Foundation

React 19 + Vite + TypeScript + Tailwind CSS

## Getting Started

```bash
npm install
npm run dev        # dev server
npm run build      # production build
npx tsc --noEmit   # type-check only
```

## Tech Stack
- React 19 · Vite · TypeScript · Tailwind CSS v3 · React Router v6
